# Trrip AI Itinerary Generator

An AI-powered travel planning application built using the MERN Stack.

Users can upload travel-related booking documents such as flight tickets, hotel reservations, and travel bookings. The application extracts key travel information and uses AI to automatically generate a structured travel itinerary.

## Features

- User Authentication (JWT)
- User Registration & Login
- Travel Document Upload (PDF/Image)
- AI-Based Data Extraction
- AI-Powered Itinerary Generation
- MongoDB Storage
- Itinerary History
- Shareable Itineraries
- Responsive React UI

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

- Multer

- Google Gemini API
